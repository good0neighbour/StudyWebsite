가. 스프라이트 리소스 제작 시

프레임 당 파일로 제작
프로그램에서 다루기 쉬우나 파일 로드 연산이 들어간다.

여러 프레임을 한 파일에 담아 제작
각 프레임의 너비, 높이를 일정하게 제작한다. 파일 로드의 연산이 비교적 적으나 프로그램에서 다루기 어렵다. 각 프레임의 너비, 높이를 제각각으로 제작하는 경우에는 더욱 어려워진다.

나. 개념

pixel per unit
1유닛에 몇개의 픽셀이 들어가는지에 대한 수치다.

Sprite Animation: frame by frame 애니메이션
일정 시간 간격으로 순차적으로 연속된 이미지를 보여주어 애니메이션을 만드는 것이다.

Sprite Sheet
여러 프레임이 한 파일로 만들어져 있는 스프라이트 파일이다.

Atlas
여러 이미지가 집합되어 있는 한 파일이다.
